# httpfile

使用方法
安装
go install github.com/zangdale/httpfile@latest

使用

```
httpfile -h
Usage of httpfile:
  -d string
        http file directory
  -p uint
        http server port
```

```
httpfile     
/Users/zangdale/test/go/httpfile      [::]:56095
http://0.0.0.0:56095
http://192.168.3.254:56095
http://192.168.236.86:56095
```
